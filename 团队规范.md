### 命名约定

* 对象：
  * HTTP 资源： `*Controller`
  * HTTP 客户端： `*Client`
  * 数据库访问: `*DAO`
  * 具有 `@Scheduled` 注释的类: `*Scheduler`
  * Spring 上下文配置类: `@Configuration`
  * 用于映射来自 yaml 文件的值的类：`*Properties`
* 数据/结构
  * 数据库返回的结构体: `*Entity`
  * HTTP 请求和响应的结构体: `*DTO`
  * 用于呈现 HTML 站点的结构体: `*Model`
  * 内部结构体，不使用后缀
